타일서바이벌 할인충전 가이드

이 페이지에서는 타일서바이벌 셀프충전 전에 꼭 확인해야 할 내용을 먼저 정리하고, 구글 또는 페이스북 계정 연동, 2차인증, 구글플레이게임즈 연동 불가 안내, 실제 진행 순서와 쿠폰 적용 방법까지 한 번에 설명합니다.

시작 전에 이 2가지를 먼저 확인해주세요

셀프충전 전 아래 두 가지를 먼저 확인하면 로그인 실패나 중간 멈춤 같은 문제를 대부분 줄일 수 있습니다.

중요: 개인정보와 계정 보안은 본인이 직접 관리해야 합니다. 결제 전에 로그인 방식과 보안 설정이 맞는지 먼저 확인해주세요.

1. 직접 연동된 계정인지 확인

타일서바이벌 계정은 구글 계정 또는 페이스북 계정에 직접 연동되어 있어야 합니다.

2. 2차인증이 설정되었는지 확인

연동된 구글 또는 페이스북 계정에 2차인증이 설정되어 있어야 합니다.

이용 가능한 계정 연동 방식 셀프충전은 구글 계정 또는 페이스북 계정에 직접 연동된 타일서바이벌 계정만 지원합니다.

구글플레이게임즈로 연동된 계정도 사용할 수 있나요?

아니요. 구글플레이게임즈로 연동된 계정은 현재 셀프충전을 사용할 수 없습니다. 현재는 구글 계정 또는 페이스북 계정에 직접 연동된 경우만 이용 가능합니다.

간단 확인: 구글이나 페이스북으로 직접 로그인하고 있고, 로그인할 때 추가 인증 단계가 나온다면 준비가 거의 끝난 상태입니다.

왜 2차인증이 꼭 필요한가요?

셀프충전은 로그인 과정에서 보안 확인이 필요합니다. 그래서 연동된 구글 또는 페이스북 계정에 2차인증이 설정되어 있어야 정상적으로 진행할 수 있습니다.

구글 2차인증 설정 방법

  • 구글 계정 관리 페이지에 접속합니다.
  • 보안 메뉴로 이동합니다.
  • 2단계 인증 항목을 찾습니다.
  • 휴대폰 번호, 인증 앱 등 원하는 인증 수단을 등록합니다.
  • 설정 후 정상적으로 적용되었는지 다시 확인합니다.

페이스북 2차인증 설정 방법

  • 페이스북 설정 및 개인정보 메뉴로 들어갑니다.
  • 비밀번호 및 보안 메뉴로 이동합니다.
  • 2단계 인증 관련 항목을 찾습니다.
  • 문자 인증 또는 인증 앱 등 원하는 방식으로 설정합니다.
  • 설정 후 정상적으로 적용되었는지 다시 확인합니다.
2차인증이 설정되었는지 확인하는 방법 가장 쉬운 방법은 시크릿모드에서 직접 로그인해보는 것입니다. 구글 로그인 페이지 또는 페이스북 로그인 페이지에서 본인 계정으로 로그인을 시도했을 때, 휴대폰 인증, 이메일 인증, 인증 앱 승인 요청이 나오면 2차인증이 적용된 상태입니다.
확인 기준: 휴대폰 인증, 이메일 인증, 인증 앱 승인 요청 중 하나라도 나오면 2차인증이 적용된 것입니다.

실제 셀프충전 진행 순서

계정 조건과 2차인증이 준비되어 있다면, 이후 순서는 크게 어렵지 않습니다. 아래 순서대로 따라가면 됩니다.

충전 페이지

타일서바이벌 할인충전 페이지 바로가기

검색창에 #s4qU 를 입력하면 결제 전에 추가 쿠폰을 적용할 수 있습니다.

너무 걱정하지 마세요. 조건만 맞다면 대부분은 아래 순서대로 그대로 진행하면 셀프충전을 마칠 수 있습니다.

1. 단일 패키지 선택

원하는 패키지를 선택한 뒤 Top-up Now를 누릅니다.

2. 여러 패키지 담기

여러 개를 함께 구매하고 싶다면 장바구니 버튼으로 원하는 패키지를 추가합니다.

3. 결제 진행

결제 수단을 카드 또는 PayPal 중 선택한 뒤 Pay Now를 누릅니다. 좌측은 셀프충전, 우측은 상담원 충전 방식입니다. 반드시 셀프충전을 선택해주세요.

4. 연동 계정으로 로그인

셀프충전을 선택하면 로그인 화면이 표시됩니다. 대부분은 구글 로그인이 가장 편합니다.

  • 구글 계정을 입력합니다.
  • 비밀번호를 입력합니다.
안내: 비밀번호가 헷갈린다면 미리 한 번 확인한 뒤 진행하는 것이 좋습니다.

5. 장바구니 금액 결제

장바구니에 담긴 총 금액을 확인한 뒤 결제를 완료합니다.

6. 결제 완료 확인

결제가 완료되면 완료 화면이 표시됩니다. 필요하면 Got it을 누르면 됩니다. 이제 충전이 완료된 상태입니다.

7. 게임 내 반영 확인

타일서바이벌에 접속해서 구매한 패키지가 정상적으로 들어왔는지 확인해주세요.

자주 묻는 질문

셀프충전 전에 가장 많이 헷갈리는 내용을 먼저 정리했습니다.

왜 셀프충전에서 막히는 사람이 많은가요?

대부분은 결제 문제가 아니라 계정 연동 방식이 맞지 않거나 2차인증이 설정되지 않은 경우입니다.

구글플레이게임즈 연동 계정은 사용할 수 없나요?

네. 현재는 구글 계정 또는 페이스북 계정에 직접 연동된 경우만 셀프충전을 지원합니다.

2차인증이 켜져 있는지 어떻게 확인하나요?

시크릿모드에서 로그인했을 때 휴대폰, 이메일, 인증 앱 등의 추가 인증 단계가 나오면 2차인증이 적용된 상태입니다.

쿠폰은 어디에서 입력하나요?

충전 페이지 검색창에 #s4qU 를 입력한 뒤 결제를 진행하면 됩니다.

중간에 막히면 어떻게 해야 하나요?

계정 연동 방식과 2차인증 상태를 먼저 다시 확인한 뒤, 그래도 헷갈리면 디스코드 지원 채널에서 문의해주세요.

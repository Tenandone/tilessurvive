업그레이드 재료

SSR 영웅은 전설 영웅 조각 또는 해당 영웅 조각으로 성급 업그레이드가 가능합니다.

표 보는 법

각 성급 행에서 티어별 필요 조각 수를 확인하고, 맨 오른쪽 Total에서 해당 구간 누적 수량을 바로 확인하면 됩니다.

진급 조건

시즌 진급 영웅은 시즌 플레이 도중 해금되며, 기존 에픽 SR 영웅 10성이 진급 재료로 필요합니다.

조각 소모 규칙

진급 이후 성급 업그레이드에는 에픽 영웅 조각 또는 해당 영웅 조각이 필요하며, 일반 구간보다 소모 조각 수가 2배 적용됩니다.

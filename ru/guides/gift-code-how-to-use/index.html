Как ввести gift code в Tiles Survive

Gift code в Tiles Survive вводится не на отдельной официальной веб-странице, а прямо внутри игры. На этой странице собраны текущие активные коды, бессрочные коды, которые можно использовать постоянно, и уже завершённые истёкшие коды.

Как ввести gift code в Tiles Survive

Если вы вводите код впервые, просто идите по шагам ниже.

  1. Зайдите в игру и откройте меню Профиль → Настройки → Обмен пакета.
  2. Введите gift code Tiles Survive точно в том же регистре.
  3. После регистрации проверьте почту или экран получения наград.

Что быстро проверить, если код не работает

Проверьте пункты ниже по порядку.

  • Проверьте написание кода и регистр, введены ли они точно так же.
  • Сначала проверьте, не истёк ли код.
  • Проверьте ограничение один раз на аккаунт, если оно есть.

FAQ по gift code Tiles Survive

Часто задаваемые вопросы.

Где вводить код в Tiles Survive?
Код вводится в игре через меню Профиль → Настройки → Обмен пакета.
Как различать бессрочные и временные коды?
Постоянные коды помечаются отдельно как бессрочные, а коды с ограничением по времени отображаются как активные или истёкшие.
Что проверять, если код не применяется?
Сначала проверьте регистр, срок действия, ограничение один раз на аккаунт и минимальные требования по уровню.
Можно ли посмотреть истёкшие коды?
Да. В нижней части этой страницы можно посмотреть историю уже истёкших кодов.

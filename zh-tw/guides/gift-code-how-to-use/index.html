Tiles Survive 禮包碼兌換方法

Tiles Survive 的禮包碼不是在專用頁面輸入，而是直接在遊戲內兌換。 本頁會把目前有效代碼、可長期使用的常駐代碼，以及已結束的過期代碼分開整理。

Tiles Survive 禮包碼兌換方法

第一次輸入時，照下面順序操作就可以。

  1. 進入遊戲後，前往 個人資料 → 設定 → 禮包兌換 選單。
  2. 將 Tiles Survive 禮包碼包含大小寫完整輸入。
  3. 完成兌換後，到信箱或獎勵領取畫面確認道具。

代碼無法使用時的快速檢查

請依序確認下面幾項。

  • 代碼拼寫與大小寫是否輸入正確。
  • 先確認是不是已過期的代碼。
  • 確認該代碼是否有每個帳號僅能使用一次的限制。
  • 部分代碼可能也有最低等級條件。

已過期禮包碼記錄

已結束使用期限的代碼會保留在這裡作為記錄。

Tiles Survive 禮包碼 FAQ

這裡整理了最常見的問題。

Tiles Survive 的代碼要在哪裡輸入？
請在遊戲內的個人資料設定中的禮包兌換選單輸入。
常駐代碼和限時代碼要怎麼區分？
可長期使用的代碼會另外標示為常駐代碼，有期限的代碼則會區分為有效代碼或已過期代碼。
代碼不能使用時要先檢查什麼？
請先確認大小寫、是否已過期、是否有帳號單次使用限制，以及是否有最低等級條件。
也可以查看已過期代碼嗎？
可以，本頁下方也整理了已過期代碼記錄。
